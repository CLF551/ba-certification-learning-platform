## Roadmap Structure

Since this is a BA portfolio project (not a funded commercial product), the roadmap phases are:

| Phase | Focus | Timeline |
|-------|-------|----------|
| **Phase 0: Foundation** | Problem discovery, requirements, design | Weeks 1-3 |
| **Phase 1: MVP** | Core prototype with syllabus map + retrieval + basic practice | Weeks 4-6 |
| **Phase 2: Validation** | User testing, iteration | Weeks 7-8 |
| **Phase 3: Intelligence** | AI features, adaptive learning | Post-validation |
| **Phase 4: Platform** | Multi-certification, cloud sync | Post-validation |

## Phase 1: MVP Build (Weeks 4-6)

### Goal

Build a clickable prototype that demonstrates the core value proposition.

### Activities

| Week | Task | Output |
|------|------|--------|
| 4 | Draw wireframes in Draw.io based on wireframe descriptions | Static wireframe screens |
| 4 | Connect wireframes into interactive flow | Clickable prototype |
| 5 | Add real AI-901 content for 2 modules | Authentic content demo |
| 5 | Implement search + Quick Look behavior in prototype | Interactive demo |
| 6 | Implement quiz flow (3-5 sample questions) | Demonstration of assessment |
| 6 | Internal walkthrough + fix flow issues | Polished prototype |

### Key Decisions

| Decision | Choice | Rationale |
|----------|--------|-----------|
| Tool | Draw.io (or Figma if learned) | User already familiar with Draw.io |
| Fidelity | Medium-fidelity (grayscale, real content) | High enough for user testing, low enough for iteration speed |
| Content source | Manually written from Microsoft Learn | Ensures accuracy; demonstrates BA role in content curation |

---

## Phase 2: Validation (Weeks 7-8)

### Goal

Test the prototype with real users and iterate.

### Activities

| Week | Task | Output |
|------|------|--------|
| 7 | Recruit 3-5 test users (peers, online study groups) | Test session schedule |
| 7 | Conduct moderated prototype testing | User feedback + recordings |
| 8 | Analyze feedback, identify patterns | Test report |
| 8 | Prioritize and implement changes | Iterated prototype v2 |

### Test Script (High-level)

1. "You want to study the AI-901 certification. Show me how you would start."
2. "You forgot what 'Azure Cognitive Services' covers. Find it."
3. "You finished Module 1. Show me how you would test your understanding."
4. "What do you think of this screen? What would you change?"
5. "Would you use this tool for your certification study? Why or why not?"
